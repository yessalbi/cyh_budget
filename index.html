모든 기기에서 데이터를 동기화하려면
Firebase 설정이 필요해요 (무료, 5분 소요)
1
Firebase 프로젝트 생성
console.firebase.google.com 접속 → 프로젝트 추가 → 이름 입력 (예: my-budget) → Google Analytics 끄기 → 프로젝트 만들기
2
Realtime Database 활성화
왼쪽 메뉴 빌드 → Realtime Database → 데이터베이스 만들기 → 테스트 모드로 시작 → 데이터베이스 URL 복사

URL 예시: https://my-budget-xxxxx-default-rtdb.firebaseio.com
3
웹 앱 등록 후 API Key 복사
프로젝트 개요 → 웹 앱 추가(</>) → 앱 닉네임 입력 → Firebase SDK 설정에서 apiKey와 databaseURL 복사
동기화 없이 사용하려면 로컬 모드로 시작
